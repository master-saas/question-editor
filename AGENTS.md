# question-editor

Single-file Python GUI application for editing JSON question files.

## Entry Point

```bash
python question_editor.py
```

## Requirements

- Python 3.x
- tkinter (stdlib for Python)
- Pillow (`pip install Pillow`)

## Usage

Run the script inside an `output/{year}` folder containing a `questions/` subdirectory with question folders.

## Keyboard Shortcuts

- `Ctrl+S` - Save current question
